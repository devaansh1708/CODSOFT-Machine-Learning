# CODSOFT-Ma# CODSOFT Machine Learning Internship

Repository containing solutions for the Machine Learning internship tasks at CodSoft.

## Projects

### 1. SMS Spam Prediction (`01_SMS_Spam_Prediction`)
- **Objective**: Build an AI model that classifies SMS messages as spam or ham.
- **Models Used**: Naive Bayes, Logistic Regression, TF-IDF Vectorization.

### 2. Customer Churn Prediction (`02_Customer_Churn_Prediction`)
- **Objective**: Predict customer churn using historical behavior and subscription data.
- **Models Used**: Logistic Regression, Random Forest, Gradient Boosting.

### 3. Credit Card Fraud Detection (`03_Credit_Card_Fraud_Detection`)
- **Objective**: Detect fraudulent credit card transactions handling high class imbalance (~0.58% fraud).
- **Models Used**: Decision Tree Classifier, Random Forest Classifier (Class Weighting / Resampling).
- **Key Metrics**: High Recall (~0.95) & ROC-AUC (~0.965).
